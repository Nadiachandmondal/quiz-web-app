🎓 QuizApp - Interactive Quiz Web Application

QuizApp is a responsive and modern web-based quiz platform designed to make learning fun and engaging. Users can explore topic-based quizzes, get instant feedback, and enjoy a smooth UI experience across all devices.

This project is created mainly for learning and practice purposes, focusing on improving frontend development skills using HTML, CSS, and JavaScript.

✨ Features
🎯 Topic-wise quiz system based on subjects and levels
📱 Fully responsive design (mobile, tablet, desktop)
🎨 Modern and clean UI with smooth animations
📊 Instant feedback and score display after quiz completion
🧑‍🎓 Easy navigation with user-friendly interface
💬 About Us page with team member details
📩 Feedback form integration using email submission
🛠️ Tech Stack
HTML5
CSS3 (Responsive Design + Animations)
JavaScript (Vanilla JS)
